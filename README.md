• AI ML Task 2 - Introduction to AI and ML

1. What is Artificial Intelligence?
Artificial Intelligence (AI) is the ability of machines to perform tasks that normally require human intelligence.

2. What is Machine Learning?
Machine Learning (ML) is a subset of AI that enables machines to learn from data and improve without being explicitly programmed.

3. What is Deep Learning?
Deep Learning is a subset of ML that uses neural networks with multiple layers to learn complex patterns.

4. Types of AI
- Narrow AI
- General AI
- Super AI

5. Types of Machine Learning
- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning
