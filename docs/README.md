# Introduction to Artificial Intelligence & Machine Learning Fundamentals

## Project Overview

This project is part of the AI & Machine Learning Internship (Week 1 - Task 2).

The purpose of this project is to understand the basic concepts of Artificial Intelligence (AI) and Machine Learning (ML), their different types, real-world applications, and ethical considerations.

---

## Topics Covered

- Introduction to Artificial Intelligence
- Introduction to Machine Learning
- Difference between AI and ML
- Types of AI
- Types of Machine Learning
- Real-world AI/ML Use Case
- AI Ethics and Responsibility
